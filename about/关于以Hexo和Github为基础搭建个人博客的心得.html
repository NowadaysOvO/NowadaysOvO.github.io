关于以Hexo和Github为基础搭建个人博客的心得

  • 第一次搭建个人博客网站的朋友也可以参考该篇 作者：最喜欢下雪天

第一章 Github的注册

详细步骤：

  • 进入GitHub官网，点击左侧Github即可进入
  • 点击sign up进行Github账号的注册（最好使用Chrome内核的浏览器）
  • 接着进行注册，不会的单词这些可以使用翻译网页或软件

第二章 安装Git

详细步骤：

  • 进入Git官网
  • 点击Downloads进行下载
  • 系统类型根据你的电脑系统进行选择（看好自己的电脑系统是32位或64位)
  • 别安装到系统盘（可以自己在其他磁盘创建一个目录），点击next
  • 保持默认，新手小白一直按next就好，后期更熟悉Git之后可以尝试自己摸索
  • 如果在安装第一步勾选了On the Desktop的话就可以看到跟我一样的桌面图标，没有也无伤大雅，鼠标右键桌面即可看到Gti Bush Here的图标和字样，点击打开就好
